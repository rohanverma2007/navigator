Navigator

Your Services Dashboard

A lightweight, self-hosted service dashboard with dynamic config and built-in status monitoring. Everything runs in one container.

Super Lightweight

Only ~2.2 MB for the API and site. The HTML is just ~7KB - fits in one packet.

Dynamic Config

Add entries to service.json and the site updates automatically without restart

Built-in Status API

Pings each service to check availability. Everything runs in one container (~56KB total)

Dark/Light Mode

Toggle between themes with a built-in mode switcher

Docker Ready

Simple setup with Docker Compose or Podman

Minimal & Clean

Designed to be as lightweight and simple as possible
